## 7. 风险与待验证项

### 7.1 技术风险

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 语雀 API 限流 | 数据同步慢 | 加延迟、分批同步 |
| gno MCP 稳定性 | 检索失败 | 降级到 grep 搜索 |
| LLM 推送判断不准 | 噪音或漏推 | 人工反馈调优 |
| 用户画像准确性 | 推荐质量差 | 迭代优化 prompt |

### 7.2 待验证项

| 问题 | 验证方式 |
|------|----------|
| AstrBot Docker 内能否访问 `/home/admin/yuque-docs` | 部署测试 |
| gno MCP 是否支持 AstrBot 调用 | 集成测试 |
| 伙伴推荐算法效果 | 小规模测试 |
| 匿名提问箱是否需要审核机制 | 与 CAC 讨论 |
