# NovaBot - 你的学习成长伙伴

> NOVA 社团智能助手，让每一次学习都被看见

---

## 什么是 NovaBot？

NovaBot 是 NOVA 社团的智能助手，基于语雀知识库构建。它不只是"知识库检索工具"，而是你的**学习成长伙伴**。

**核心理念**：
- 🎯 记住你的学习轨迹
- 🤝 连接志同道合的人
- 💡 主动关心你的成长

---

## 你可以这样用它

### 新人入社

> "社团在做什么？有哪些兴趣组？"

```
/novabot 社团有哪些兴趣组？
/kb guide NOVA活动方案
/partner
```

NovaBot 会帮你快速了解社团，找到适合你的兴趣组和学习伙伴。

---

### 日常学习

> "我想学爬虫，怎么入门？" "这个问题怎么解决？"

```
/path 爬虫
/ask 如何处理验证码？
帮我搜索爬虫相关的文档
```

NovaBot 会为你规划学习路径、解答问题、推荐相关文档。

---

### 找学习伙伴

> "谁也在学 Python？有没有人可以一起做项目？"

```
/partner Python
/collab find AI应用开发
```

NovaBot 会根据兴趣匹配，推荐学习伙伴或协作对象。

---

### 查看学习成果

> "我最近学了什么？还差什么知识？"

```
/profile
/progress
/gap
```

NovaBot 记录你的学习轨迹，让你看到自己的成长。

---

### 订阅更新推送

> "我想及时知道某个知识库的更新"

```
/subscribe repo 爬虫学习笔记
/subscribe author 张三
/subscribe all
```

NovaBot 会在文档更新时，智能判断是否值得推送，并将变更要点摘要发送给你。

## 智能功能一览

### 知识检索

| 指令 | 说明 |
|:-----|:-----|
| `/kb` | 列出所有知识库 |
| `/kb <知识库>` | 查看知识库概览（文档数、贡献者、最近更新） |
| `/kb <知识库> <问题>` | 在指定知识库范围内问答 |
| `/kb guide <知识库>` | 生成知识库新人导航 |
| `/kb updates <知识库>` | 查看知识库近期更新动态 |
| `/card <主题>` | 生成知识卡片 |

> 💡 **提示**：全局搜索文档可直接用自然语言，如"帮我找爬虫教程"

### 学习成长

| 指令 | 说明 |
|:-----|:-----|
| `/path <领域>` | 个性化学习路径（需绑定） |
| `/gap [领域]` | 分析学习缺口（需绑定） |
| `/progress` | 学习进度报告（需绑定） |
| `/weekly` | 本周知识周报 |

> 💡 标注"需绑定"的功能需要先执行 `/bind` 绑定语雀账号

### 社交连接

| 指令 | 说明 |
|:-----|:-----|
| `/partner [主题]` | 学习伙伴推荐（需绑定） |
| `/collab find <主题>` | 协作伙伴推荐（需绑定） |
| `/trajectory [成员]` | 查看活动轨迹（需绑定） |

> 💡 社交功能需要绑定账号才能使用

### 订阅与推送

| 指令 | 说明 |
|:-----|:-----|
| `/subscribe` | 查看我的订阅列表 |
| `/subscribe repo <知识库名>` | 订阅知识库更新 |
| `/subscribe author <作者名>` | 订阅作者更新 |
| `/subscribe all` | 订阅全部更新 |
| `/unsubscribe [订阅ID]` | 取消订阅 |

**智能推送特性**：
- 文档更新时自动推送通知
- LLM 智能判断更新是否值得推送
- 生成变更要点摘要，快速了解更新内容
- 推送消息示例：
  ```
  张三 更新了《爬虫入门指南》（知识库：爬虫学习笔记）

  主要变更如下：
  1. 新增 Selenium 自动化章节
  2. 更新了反爬虫应对策略
  3. 修正了代码示例中的错误

  原文：https://www.yuque.com/...
  ```

## 快速开始

### 第一步：绑定账号

```
/bind <你的语雀用户名>
```

绑定后，NovaBot 才能记录你的学习轨迹，提供个性化服务。

### 第二步：生成画像

```
/profile refresh
```

NovaBot 会分析你在语雀的文档，生成你的学习画像（兴趣领域、技能水平等）。

### 第三步：开始探索

- 新人：`/kb guide NOVA活动方案`
- 找方向：`/path <你感兴趣的领域>`
- 找伙伴：`/partner`
- 有问题：`/ask <你的问题>`

## 技术特性

- **语雀集成**：自动同步知识库文档，Markdown + YAML frontmatter 存储
- **RAG 检索**：语义化搜索，找到真正相关的内容
- **用户画像**：基于文档关键词提取，了解你的兴趣和水平
- **Agent 智能体**：理解自然语言，自动选择工具，个性化回复
- **长期记忆**：记住对话历史，追踪学习进度
- **协作网络**：分析成员关系，智能推荐伙伴
- **智能推送**：LLM 判断更新价值，生成摘要，精准推送给订阅者
